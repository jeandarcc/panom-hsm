## What HSM Is

HSM brings route behavior, query behavior, policy, backend authorization, and browser URL behavior into one hierarchical state model.

## Why It Exists

- to connect routing logic to a state model
- to make query parameters typed and controlled
- to bring frontend and backend policy into the same semantic layer
- to centralize advanced behavior like host-aware navigation
